# JavaSpring

## 설치방법

1. STS 설치

   => java -jar sp[Tab키누르기]

2. lombok 설치

   => java -jar lom[Tab키누르기]

3.  컨트롤러 클래스 생성

   => 메소드(@GetMapping / @RequestMapping)

   @ResponseBody 또는 @RestController

   JSON 또는 HTML

  > html에게 전달 및 결과 호출

- ### @restcontroller

  > json형식이나 값 자체를 출력
